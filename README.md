# 👋 rar.vg

## ❓ FAQ
### 🤔 What's this?
A place where you can display and expand your online presence, by designing your own dynamic website.

## How to use?
The **usual**.

### 💻 Run development server
```shell
npm start
```
### 👷‍♂️ Build
```shell
npm run build
```
